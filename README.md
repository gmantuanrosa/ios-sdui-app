# iOS SDUI App

A Server-Driven UI (SDUI) iOS application.

## Overview

This project implements a server-driven UI architecture for iOS, allowing dynamic UI updates without requiring app releases.

## Requirements

- iOS 14.0+
- Xcode 13.0+
- Swift 5.5+

## Getting Started

1. Clone the repository
```bash
git clone https://github.com/gmantuanrosa/ios-sdui-app.git
cd ios-sdui-app
```

2. Open the project in Xcode
```bash
open ios-sdui-app.xcodeproj
# or if using CocoaPods/SPM
open ios-sdui-app.xcworkspace
```

3. Build and run the project

## Architecture

This app follows a Server-Driven UI pattern where the UI components and layouts are defined by the backend and rendered dynamically on the client.
